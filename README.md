# CatchButton

<img width="799" height="481" alt="스크린샷 2026-03-12 121309" src="https://github.com/user-attachments/assets/275e654e-62b0-4504-8886-f7ba5b8f15f7" />

버튼에 커서를 가져다대면 버튼이 자동 무작위 위치로 이동하게 됨. 처음엔 커서를 가져다대면 움직이게끔 하는 버튼을 찾지 못해서 클릭해야 움직이게끔 만들었지만 MouseEnter버튼을 발견해서 해결됨.

- 어려웠던 점
: 버튼 클릭이 아니라 커서를 가져다대면 바로 버튼 위치가 바뀌는데 어떻게 +점수가 나오는지 ...?
-> 커서가 버튼 내로 들어간지 0.3초 안에 클릭해야만 성공한 것으로 간주하고, 커서로 0.3초 내에 클릭하지 못하면 실패한 것으로 간주함. 그리고 클릭 성공하는 행위의 조건을 추가함. 버튼 아무데나 누르는 것이 아닌 버튼의 50%영역만 성공 영역으로 줄임.
